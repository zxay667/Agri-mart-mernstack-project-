Next Steps

1. Keep MongoDB Running

Make sure to keep the MongoDB server running while using the application.

2. Install Dependencies

To run the full application, you need to install the dependencies:

cd agrimart/backend
npm install

cd agrimart/frontend
npm install

3. Start the Application

Once the dependencies are installed, you can start the application:

cd agrimart/backend
npm run server

cd agrimart/frontend
npm start

4. Access the Application

Open your browser and go to: http://localhost:3000
