AgriMart MongoDB Setup Guide

This guide will help you set up MongoDB for the AgriMart application.

Step 1: Start MongoDB

First, make sure MongoDB is running on your system.

  1. Run the start-mongodb.bat file in the AgriMart directory
  2. This will start the MongoDB service or the MongoDB server directly
  3. Keep the terminal window open while using the application

Step 2: Install MongoDB Compass

MongoDB Compass is a graphical user interface for MongoDB that makes it easy to work with your database.

  1. Download MongoDB Compass from: https://www.mongodb.com/products/compass
  2. Install the application following the on-screen instructions
  3. Launch MongoDB Compass

Step 3: Connect to MongoDB

Connect to your local MongoDB instance using MongoDB Compass.

  1. In MongoDB Compass, use the following connection string: 
    mongodb://localhost:27017
  2. Click "Connect"

Step 4: Create the AgriMart Database

  1. In MongoDB Compass, click "Create Database"
  2. Enter the following information:
    • Database Name: agrimart
    • Collection Name: users
  3. Click "Create Database"

Step 5: Create Additional Collections

  1. In the agrimart database, create the following collections:
    • products
    • orders
  2. To create a new collection:
    • Click on the agrimart database
    • Click "Create Collection"
    • Enter the collection name
    • Click "Create Collection"

Step 6: Add Sample Data

Add a Sample Admin User

  1. Click on the users collection
  2. Click "Add Data" > "Insert Document"
  3. Enter the following JSON: 
    {
  "name": "Admin User",
  "email": "admin@example.com",
  "password": "$2a$10$ij4H1fUVZ.Yx9yrwuXKOOOKQedlRiJ8zt.Mw0Yh6l.K04tVEJZUcW",
  "isAdmin": true
}
  4. Click "Insert"

Add a Sample Product

  1. Click on the products collection
  2. Click "Add Data" > "Insert Document"
  3. Enter the following JSON (replace [ADMIN_USER_ID] with the ObjectId of the admin user you created): 
    {
  "name": "Premium Garden Trowel",
  "image": "/images/trowel.jpg",
  "description": "High-quality stainless steel garden trowel with ergonomic handle.",
  "brand": "GardenPro",
  "category": "Tools",
  "price": 19.99,
  "countInStock": 10,
  "rating": 4.5,
  "numReviews": 12,
  "user": ObjectId("[ADMIN_USER_ID]"),
  "reviews": []
}
  4. Click "Insert"

Step 7: Run the AgriMart Application

  1. Start the backend: 
    cd agrimart/backend
npm run server
  2. Start the frontend: 
    cd agrimart/frontend
npm start
  3. Access the application at: http://localhost:3000
  4. Log in with:
    • Email: admin@example.com
    • Password: 123456

Congratulations!

You have successfully set up MongoDB for the AgriMart application.

Troubleshooting

If you encounter issues:

  • MongoDB Connection Issues: Make sure the MongoDB service is running
  • Application Startup Issues: Make sure all dependencies are installed
  • Database Access Issues: Check if MongoDB is accessible at localhost:27017
